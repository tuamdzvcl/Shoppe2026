Cấu hình Open API Shopee

Lấy App ID và App Secret từ trang Shopee Affiliate Dashboard (Open API). Dữ liệu này được lưu cục bộ trên trình duyệt của bạn, đảm bảo an toàn.
Lưu ý: Công cụ này gửi request trực tiếp đến Shopee Open API từ trình duyệt. Nếu gặp lỗi CORS (Cross-Origin Resource Sharing), bạn cần cấu hình một Backend Proxy hoặc extension chặn CORS để sử dụng nội bộ.
